# Duval Triangle Visual
A Power BI Custom Visual implementing the Duval Triangle method (Methods 1, 4, and 5) for transformer DGA diagnostics.

## 📥 Download
Navigate to the `dist/` folder in this directory to download the compiled `.pbiviz` file for use in Power BI Desktop.

## 💻 Development
* Install dependencies: `npm install`
* Start local server: `pbiviz start`
* Package visual: `pbiviz package`
