# Duval Pentagon Visual
A Power BI Custom Visual implementing the advanced Duval Pentagon method for complex fault classification.

## 📥 Download
Navigate to the `dist/` folder in this directory to download the compiled `.pbiviz` file for use in Power BI Desktop.

## 💻 Development
* Install dependencies: `npm install`
* Start local server: `pbiviz start`
* Package visual: `pbiviz package`
