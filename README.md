# 📊 Engineering Diagnostics: Power BI Custom Visuals

A collection of specialized, open-source data visualizations for **Transformer Diagnostics** and **Dissolved Gas Analysis (DGA)**. These Power BI Custom Visuals implement complex geometric algorithms to translate raw DGA gas concentrations into actionable maintenance insights directly within your reports.
